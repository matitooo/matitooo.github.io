Income Prediction using different Machine Learning approaches

June 2025 · Mattia D'Agostini
Download
  • Presentation
  • Code
Abstract

For the final project of the SoSe25 Machine Learning and Intelligent Data Analysis course, I analyzed different Machine Learning approaches to address Kaggle’s Income Prediction Challenge, which consists in predicting whether an individual earns more than $50K per year based on demographic and sociological data.

I began with an exploratory data analysis and an evaluation of possible preprocessing strategies. Subsequently, I implemented several Machine Learning models — including Random Forest, SVM, and Logistic Regression — and compared their performance using multiple evaluation metrics. Finally, I used the trained models to predict income labels for the previously unseen subjects.

A short presentation is provided alongside the project notebook.
